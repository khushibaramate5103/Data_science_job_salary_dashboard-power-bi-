# Project Overview

This project presents an interactive data visualization dashboard that analyzes global Data Science job salaries based on different factors such as experience level, company size, employment type, job title, and employee residence.

The dashboard helps identify patterns in salary distribution across different roles and regions in the data science industry.

The dashboard was created to demonstrate data analytics and visualization skills using Power BI.

# 🎯 Project Objectives

Analyze average salaries in the data science field

Understand how experience level impacts salary

Compare salaries across different employment types

Identify geographical salary trends

Explore company size distribution

Visualize top countries where employees reside
